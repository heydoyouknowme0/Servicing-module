# Neutralinojs Vite React Typescript project



# Install

```
git clone https://github.com/natewong1313/neutralinojs-vite-react-ts.git
cd neutralinojs-vite-react-ts
npm install
npx neu update
```

# Usage

Start the app with hot reload

```
npm start
```

Build the app

```
npm run build
```
